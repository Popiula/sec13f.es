## Por donde empezar

Lo primero que necesitas es instalar y cargar el paquete `devtools` desde CRAN:

```r
install.packages("devtools")
library(devtools)
```
Despues, instala el paquete `sec13f.es` desde GitHub:

```r
install_github("Popiula/sec13f.es")
library("sec13f.es")
```
